# DOM Longshot

一个无构建依赖的 Chrome Manifest V3 扩展。点击扩展按钮后选择页面元素，扩展会分段截取超出视口的部分并导出为一张 PNG 长图。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`。
2. 开启右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目目录。

## 使用

1. 在普通 HTTP 或 HTTPS 页面点击扩展图标。
2. 点击“选择页面元素”。
3. 移动鼠标预览范围，点击目标元素开始截图。
4. 等待页面自动滚动并下载 PNG；按 `Esc` 可退出选择模式。

## 当前范围

- 支持位于普通文档流、尺寸超出当前视口的元素。
- 支持选择同域及跨域 iframe 内的 DOM，并逐层换算截图坐标。
- 支持截图后恢复 iframe 各级父页面及滚动容器的位置。
- 支持选中带独立滚动条的 DOM，并按 `scrollWidth`、`scrollHeight` 截取完整内容。
- 支持横向和纵向分段，截图结束后恢复原滚动位置。
- 超大图片会自动等比缩小，以规避浏览器 Canvas 尺寸和内存限制。
- 页面滚动期间，懒加载内容有约 600ms 的加载时间。

扩展需要全部网站访问权限，才能把元素选择脚本注入跨域 iframe。iframe 外框和被选中的独立滚动容器需要能够完整放入顶层视口；暂不支持经过 CSS transform/zoom 变换的 iframe。沙盒禁止脚本的 iframe、虚拟列表、DRM 视频、持续变化的 Canvas/WebGL，以及复杂 sticky/fixed 子元素仍不保证效果。
